## Vanilla Weather App

Have a look at my little weather app with a simple but clean UI.
[You can view it here](https://vanilla-weather-app-2020-v1.netlify.app).

## How it's made

- HTML
- CSS
- JavaScript
- Bootstrap
- Ajax - Axios
- Open Weather Api
- World Time Api
